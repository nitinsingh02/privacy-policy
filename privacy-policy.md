# Privacy Policy

**Last updated: \[Date\]**

This Privacy Policy applies to the *Screen Mirroring App* ("we", "our",
"us"). This document explains how our app handles user information,
permissions, and privacy.

## 1. Overview

Screen Mirroring App enables users to:

-   Cast their device screen\
-   Paint over the screen\
-   Record videos

We prioritize user privacy and do not collect any personally
identifiable information.

## 2. Information We Collect

### No Personal Data Collection

Our app does **not** collect, store, or share any personal information
or account data from users.

## 3. Permissions Used in the App

To provide core screen mirroring, screen drawing, and video recording
functionalities, the app requires the following permissions:

### Required Permissions & Purpose

  -----------------------------------------------------------------------
  Permission                              Purpose
  --------------------------------------- -------------------------------
  INTERNET                                For screen casting connection

  ACCESS_NETWORK_STATE                    To check network status for
                                          casting

  ACCESS_WIFI_STATE                       To detect Wi-Fi status for
                                          mirroring

  FOREGROUND_SERVICE                      To run mirroring in background
                                          safely

  POST_NOTIFICATIONS / NOTIFICATION       For displaying mirroring
  PERMISSION                              notifications

  SYSTEM_ALERT_WINDOW / SMART ALERT       To show overlay for screen
  WINDOW                                  drawing

  READ/WRITE EXTERNAL STORAGE             To save recorded videos

  RECORD_AUDIO                            To capture audio during screen
                                          recording

  CAMERA                                  If the user chooses camera
                                          input during recording
  -----------------------------------------------------------------------

We only request permissions **when needed**, and only for the purpose
described above.

## 4. Third-Party Services Used

Our app uses the following third-party libraries:

-   LocalBroadcastManager (1.1.0) -- For broadcasting local events
    inside the app\
-   nanohttpd (2.3.1) -- For enabling mirroring functionality through a
    lightweight HTTP server

These libraries may process technical data required for app
functionality but **do not collect personal data**.

## 5. No Account Creation Required

Users are **not required to create an account** or sign in.\
We do not store usernames, passwords, or personal identifiers.

## 6. Children's Privacy

Our app does not target children under 13 and does not collect any
personal information from minors.

## 7. Data Security

Even though we do not collect personal information, we ensure:

-   Secure handling of permissions\
-   No unauthorized data sharing\
-   No data transmission to external servers

All processing happens locally on the device.

## 8. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. Any changes will be
posted on this page. Please review it periodically.
